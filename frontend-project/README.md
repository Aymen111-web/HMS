# Hospital Management System (HMS) - Frontend Project

This is the frontend project for the Hospital Management System (HMS) built primarily with JavaScript.

## Features

- User authentication and role-based access.
- Patient management (add, edit, view, discharge).
- Doctor management and scheduling.
- Appointment booking and tracking.
- Dashboard with analytic statistics.
- Responsive design for various devices.

## Technologies Used

- JavaScript (main codebase)
- CSS/HTML
- API integration (RESTful)
- Other minor dependencies

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Aymen111-web/HMS.git
   ```

2. **Navigate to the `frontend-project` directory:**
   ```bash
   cd HMS/frontend-project
   ```

3. **Install dependencies:**
   Install required packages (if using npm, yarn or another package manager).

   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm start
   ```

5. **Open the app in your browser:**
   Usually at [http://localhost:3000](http://localhost:3000) or the port shown in your terminal.
